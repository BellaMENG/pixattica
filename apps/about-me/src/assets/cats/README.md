Put cat photos in this folder using one subfolder per country slug.

Example:

- `src/assets/cats/japan/tokyo-cat-1.jpg`
- `src/assets/cats/japan/tokyo-cat-2.jpg`
- `src/assets/cats/france/paris-cat-1.png`

Supported file types:

- `.jpg`
- `.jpeg`
- `.png`
- `.webp`
- `.avif`

The site automatically:

- loads all photos from `src/assets/cats/<country-slug>/`
- groups them by country
- picks one random photo each time you hover that country on the map
